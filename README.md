<h1>真正的C++语言</h1>
<h1>Real C++ Language</h1>

<h2>中文版 - Chinese</h2>
<h3>这是一个由C/c混淆的C++方言，由宏构成（见real_cpp_lang.h）</h3>
<h3>编译方法（需要C++编译器）：</h3>
` g++ -O3 -std=c++11 real_cpp_lang.h <your_file> -o <your_EXE> -static `
<h4>↑ 指令自选</h4>
<h3>示例</h3>
<h4>见pi.cpp，real_cpp.exe是pi.cpp编译后的EXE文件</h4>

<h2>英文版 - English</h2>
<h3>This is a C++ dialect that obfuscates C/C++ using macros (see `real_cpp_lang.h`).</h3>
<h3>Compilation (requires a C++ compiler):</h3>
` g++ -O3 -std=c++11 real_cpp_lang.h <your_file> -o <your_EXE> -static `
<h4>↑ The options are optional.</h4>
<h3>Example:</h3>
<h4>See `pi.cpp`; `real_cpp.exe` is the EXE file compiled from `pi.cpp`.</h4>
